# Bs3harp

harpJS + browser-sync + gulp + Bootstrap3

## Usage

### Install gulpjs and some dependencies.

/Bs3harp

	$ npm install

### Run server with Livereload

/Bs3harp/public

	$ harp server

/Bs3harp

	$ npm start

### Compile source 

Compile source in /Bs3harp/public/www 

/Bs3harp/public

	$ harp compile

Compile source non-minify-html in /Bs3harp/dist

/Bs3harp

	$ gulp
